# Secure-Login-System
A secure login web application developed using Flask, SQLite, and bcrypt. The application allows users to register, log in, and securely manage sessions.
## features
- User Registration
- Secure Login
- Password Hashing (bcrypt)
- Session Management
- Logout Feature
- SQL Injection Protection
## Technologies Used
- Python
- Flask
- SQLite
- bcrypt
## Installation
```
pip install flask bcrypt
```
## Run
```
python app.py
```
